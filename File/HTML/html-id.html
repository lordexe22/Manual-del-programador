Introducción

El atributo id es usado para especificar un id único a un elemento HTML. No se puede tener más de un elemento con el mismo id en un mismo documento HTML.

Aplicación

El atributo id se usa habitualmente como punto de referencia para poder acceder al elemento que lo contiene. Esto se logra referenciando el nombre del id a la cual pertenece el objeto HTML.

Esto es particularmente util cuando se desea acceder o manipular un elemento, ya sea para aplicar estilos con CSS o alterar su comportamiento con Java Script.

Id en HTML/CSS

Para ver como trabaja un id, crearemos un elemento HTML y posteriormente se le haran modificaciones mediante CSS y JavaScripts. Para referencia un id en CSS, se usa el signo de numeral (#) más el nombre del id, finalmente se cierra entre llaves los atributos que se le asignaran a dicha clase.

Para comenzar crearemos un elemento div, le asignaremos un color de fondo verde oscuro, un color de texto blanco y un padding de 10 pixeles. No es importante entender como funciona el código en CSS, sino observar que se accede al elemento mediante el nombre de su id.


<!--Creamos un elemento div y le asignamos un id llamado color-->

<div id="color">
Contenido
</div>
/*Creamos el cuerpo del id color*/

#color {
background-color:darkgreen;
color:white;
padding:10px;
}

Contenido

Es importante notar como accedemos al elemento div mediante su id "color" y podemos modificarlo visualmente desde el código en CSS.

Id en HTML/JavaScript

En este caso, se realiza un Script que oculte el contenido del elemento que contengan la id "cambiarMensaje". Para ello pulsamos un boton que accione el Script. Bookmarks con <id> y <a>

Un bookmark nos permite hacer saltos entre distintas secciones de un mismo documento HTML. Se uso de hecho para armar la sección de "Contenido" al inicio de esta página.

Para mostrar como crear un bookmark, realizaremos un ejemplo que nos lleve a la sección de contenido de este documento.

<!--(1) Primero creamos un elemento y le asignamos un id -->
<h2 id="0"> Contenido </h2>

<!--(2) Creamos un elemento con el atributo link y le anexamos el nombre del id al que deseamos desplazarnos al hacerle click -->
<a href="#0">
<button> Ir a Contenido </button>
</a>

Cualquier tipo de contenido (botones, parrafos, etc) dentro de la etiqueta <a> nos enviara al elemento que contenga el id marcado.

Diferencias entre class e id

La diferencia fundamental entre los atributos class e id es que las clases se pueden aplicar a varios elementos al mismo tiempo dentro de un mismo documento HTML, mientras que un nombre de id es único y exclusivo para un solo elemento HTML y no se puede compartir.

Sumario

  • El atributo id se usa para especificar un id única para un elemento HTML.
  • El valor del atributo id debe ser único dentro del documento HTML.
  • El atributo id es usado por CSS y JavaScript para seleccionar o estilizar un elemento específico.
  • El nombre de un id es sensible a mayusculas y minusculas.
  • El nombre de un id debe contener al menos un caracter, no debe contener espacios en blanco y preferentemente no debe comenzar con un número.
  • El atributo id es usado también para crear bookmarks.
  • JavaScript puede acceder a un elemento con un id especifico con el metodo getElementById().
