Introducción

El elemento HTML canvas es usado para dibujar gráficos en una página web, sobre la marcha por medio de JavaScript. El elemento canvas es solamente un contenedor para gráficos y se deben dibujar los mismos con JavaScript. Para esto contamos con varios metodos para dibujar lineas, cajas, circulos, texto y agregar imagenes entre otros.

Para crear un canvas es necesario especificar un id para poder referenciarlo con JavaScript y hay que asignarle un alto y ancho para delimitar las dimensiones sobre las cuales podremos crear el gráfico. Para poder visualizar las dimensiones del mismo podemos agregarle un borde. En el siguiente ejemplo creamos un canvas totalmente vacio.

<canvas id="myCanvas" height="200px" width="200px" style="border thin black solid">

Un canvas es un lienzo en blanco, en las siguientes secciones veremos los comandos que podemos utilizar para dibujar en ellos.

NOTA: Por defecto un elemento canvas no tiene bordes ni contenido.

NOTA: Se pueden tener varios elementos canvas en un mismo documento HTML.

Aplicaciones

Como se menciono anteriormente un canvas es un elemento HTML que nos permite dibujar graficos en una página web usando comandos de JavaScript.

Dentro de las posibilidades que nos permite canvas podemos:

  • Dibujas gráficos: canvas tiene una gran cantidad de caracteristicas para la presentación gráfica de datos.
  • Animaciones: Los objetos de canvas pueden moverse. Todo es posible, desde movimientos simples de objetos hasta animaciones complejas.
  • Interactividad: canvas puede responder a eventos de JavaScript e interacciones con el usuario (teclas, mouse, botones, etc).
  • Videojuegos: los metodos de animación de canvas ofrecen muchas posibilidades de desarrollo para aplicaciones de juegos.

Primeros pasos

El primer paso es crear en nuestro documento HTML un elemento canvas. En el siguiente ejemplo creamos un canvas con el id canvas-1 y le asignamos un alto y ancho de 200px, además dibujamos un borde para poder visualizar las dimensiones del mismo.

<canvas id="canvas-1" height="200px" width="200px" style="border: thin black solid">

Lo segundo es referenciar a nuestro elemento canvas desde nuestro documento de JavaScript. Para esto creamos una variable y le asignamos una referencia hacia el elemento canvas mediante el método de HTML DOM getElementById("myCanvas");.

let myCanvas1 = document.getElementById("canvas-1");

Lo tercero que necesitamos es un objeto dibujable para el canvas. Para esto se usa la función getContext("2d"); que es el contexto de dibujo del elemento canvas y posee propiedades y métodos para dibujar sobre el mismo. Con este paso completado ya estaremos en condiciones de crear gráficos

let myContext1 = myCanvas1.getContext("2d");

NOTA: Para verificar que el navegador soporte canvas, el script puede ejecutar la función anterior de la siguiente forma:

let myCanvas1 = document.getElementById(canvas-1);

if (myCanvas1.getContext){
// Código del dibujo aquí
} else {
// Código si no es soportado aquí
}

El tercer y último paso consiste en dibujar. Para conocer las herramientas de dibujo que ofrece canvas en secciones posteriores se prueban todas ellas. Con este pequeño ejemplo ilustrativo dibujamos un cuadrado rojo en el centro del canvas.

<canvas id="canvas-id" height="200px" width="200px" style="border: thin black solid">
let myCanvas = document.getElementById("canvas-id");
let myContext = myCanvas.getContext("2d");
myContext.fillStyle = "#FF0000"; // Establecemos un color, gradiente o patrón de colores de CSS
myContext.fillRect(50, 50, 100, 100); // Establecemos la posición de origen, el ancho y el alto del rectangulo.

NOTA: Los canvas estan compuestos por una grilla de dos dimensiones cuyo origen de coordenadas (0,0) se encuentra siempre arriba a la izquierda.
