Introducción

Los comments (comentarios) en HTML son texto que no se muestran en la página web, pero ayudan a documentar el código fuente de los archivos.

Agregar comentarios

Con los comentarios podemos poner notificaciones y recordatorios en nuestro código HTML.

Para introducir un comentarios se usa la siguiente sintaxis: !<-- Esto es un comentario -->.

Aplicaciones de los comentarios

  • Podemos usar los comentarios para explicar el funcionamiento de algún fragmento del código, asi como tambien para dejar notas en comandos que no conozcamos.

  • Los comentarios pueden usarse para notificar sobre el estado del código en el documento (existencia de errores, informar obsolesencias, bugs, etc).

  • Los comentarios se usan con frecuencia para comentar fragmentos de código que no queremos ejecutar cuando estamos realizando pruebas sobre el documento.

  • Los comentarios pueden usarse hasta para dejar redactada una lista de tareas a realizar. Básicamente, podemos dejar anotada cualquier cosa que nos interese, no existe ningun tipo de restricción.

Ejemplo

<!--Esto es un comentario y no sera interpretado con el resto del código-->
<h1>Título principal de la página</h1>

Título principal de la página
