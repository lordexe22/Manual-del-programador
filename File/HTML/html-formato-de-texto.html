Elementos de formato

HTML contiene varios elementos para definir distintos tipos de texto on un significado especial.

Los elementos de formato fueron diseñados para mostrar tipos especiales de texto, estos son:

<b> - Define texto en negrita (bold) sin importancia alguna.

Texto normal - texto con <b>


<strong> - Define un tipo de texto con una importancia relevante. El texto se visualizara en negrita.

Texto normal - texto con <strong>


<i> - Define texto en cursiva (italic). Se usa usualmente para indicar terminos técnicos una frase en otro idioma, un pensamiento, etc.

Texto normal - texto con <i>


<em> - Define texto enfatizado. El contenido se mostrará tipicamente en italic. La posición de este elemento en una oración y las palabras que encierra son importantes para la interpretación de su significado. Sin embargo, no confiere importancia a su contenido.

Texto normal - texto con <em>


<mark> - Define texto que debe ser marcado o resaltado. El contenido se mostrará resaltado con un fondo amarillo. Representa un texto marcado o resaltado como referencia o anotación, debido a su relevancia o importancia en un contexto particular.

Texto normal - texto con <mark>


<small> - Define texto pequeño. Este elemento es reutilizado para representar comentarios laterales y letra pequeña, incluyendo derechos de autor y texto legal, independientemente de su estilo de presentación.

Texto normal - texto con <small>


<del> - Define texto borrado. Marca las partes de un texto o documento que han sido suprimidas o sustituidas. Se visualizara con una linea horizontal tachando el contenido.

Texto normal - texto con <del>


<ins> - Define texto insertado. Marca las partes de un texto que han sido añadidos al documento. Se visualizara con una linea horizontal subrayando el contenido.

Texto normal - texto con <ins>


<sub> - Representa un texto de subindice. Se usa en fragmentos de texto que por razones tipográficas deben visualizarse por debajo y en tamaño más pequeño que la fuente normal de texto. Se puede usar en la escritura de fórmulas.

Texto normal - texto con <sub>


<sup> - Representa un texto de superindice. Se usa en fragmentos de texto que por razones tipográficas deben visualizarse por encima y en tamaño más pequeño que la fuente normal de texto. Se puede usar en la escritura de fórmulas.

Texto normal - texto con <sup>


Notas

Bajo ninguna circunstancia se debe buscar dar estilos al contenido de la página web con estas etiquetas de formato de texto Las mismas deben usarse para darle estructura al contenido de la página, pero nunca para darle estilo, para eso se usa CSS.
