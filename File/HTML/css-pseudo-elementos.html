Introducción

Los pseudo-elementos son fragmentos de código que aplicarán estilos css en fragmentos de los elementos contenidos dentro de un documento HTML.

Sintaxis

Para usar un pseudo-elemento en CSS, luego de escribir el selector se ponen :: y a continuaciòn el nombre del pseudo-elemento de interés. En la próxima sección se listan todos los pseudo-elementos existentes.
