Introducción

El Módulo de Caja Flexible, comúnmente llamado flexbox, fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación.

Cuando describimos a flexbox como unidimensional destacamos el hecho que flexbox maneja el layout en una sola dimensión a la vez — ya sea como fila o como columna. Esto contrasta con el modelo bidimensional del Grid Layout de CSS, el cual controla columnas y filas a la vez.

Términos básicos

Para comprender facilmente el funcionamiento de los atributos referentes a flexbox es necesario conocer las diferentes partes que lo componen.

  • main axis: Es el eje primario de un contenedor flex. Este eje puede ser horizontal o vertical mediante la propiedad flex-direction.
  • cross axis: Es el eje perpendicular al main axis.
  • main size: Es el tamaño (alto o ancho) del contenedor principal del flexbox. Se edita el alto o el ancho según cual sea el main axis que se haya configurado. El tamaño puede configurarse con las propiedades width o height.
  • cross size: Es el tamaño (alto o ancho) del contenedor perpendicular al eje principal del flexbox. Se edita el alto o el ancho según cual sea el cross axis que se haya configurado. El tamaño puede configurarse con las propiedades width o height.
  • main start | main end: Los items dentro del contenedor son colocados en el eje principal, dentro del rango determinado por main start y main end.
  • cross start | cross end: Los items dentro del contenedor son colocados en el eje perpendicular al eje principal, dentro del rango determinado por cross start y cross end.
  • flex container: Contenedor que almacenara los flex-items (contenido del flexbox), el mismo se usa para establecer las dimensiones dentro de las cuales se pueden colocar los flex-items.
  • flex item: En el mismo se pone el contenido de la página.

Creando un flexbox

Para crear un flexbox, en HTML organizamos la estructura del mismo, para ello debemos declarar un elemento que trabaje como contenedor (flex-container) y que contenga uno o más elementos hijos que jueguen el roll de objetos dentro del flexbox (flex-item).

Por otro lado en CSS les daremos a la estructura creada los estilos apropiados. El único atributo obligatorio es dotar al elemento contenedor con la propiedad display y asignarle el valor flex para que los items tengan las propiedades de un elemento de tipo block o bien el valor inline-flex para que los items tengan las propiedades de un elemento de tipo inline. Luego de esto podemos modificar tanto al elemento contenedor como a sus elementos hijos.


<div class="flex-container">
<div class="flex-item">Item 1</div>
<div class="flex-item">Item 2</div>
<div class="flex-item">Item 3</div>
<div class="flex-item">y asi con el resto de items</div>
</div>
.flex-container{
display: flex;
border: medium black solid;
}

.flex-container>.flex-item{
border: medium blue solid;
background-color: green;
color: white;
}
