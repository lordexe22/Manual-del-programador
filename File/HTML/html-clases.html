Introducción

Una clase en HTML es un atributo que se usa para especificar "la clase" de un elemento HTML.

Para especificar la clase de un elemento se utiliza el atributo class y posteriormente se indica el nombre de la clase creada.

El atributo class se puede aplicar en cualquier elemento HTML y varios elementos HTML pueden compartir la misma clase.


Aplicación

El atributo class se usa habitualmente como punto de referencia para poder acceder al elemento que lo contiene. Esto se logra referenciando el nombre de la clase a la cual pertenece el objeto HTML.

Esto es particularmente util cuando se desea acceder o manipular un elemento, ya sea para aplicar estilos con CSS o alterar su comportamiento con Java Script.

Clases en HTML/CSS

Para ver como trabaja una clase, crearemos un elemento HTML y posteriormente se le haran modificaciones mediante CSS y JavaScripts. Para referencia una clase en CSS, se usa un punto (.) más el nombre de la clase, finalmente se cierra entre llaves los atributos que se le asignaran a dicha clase.

Para comenzar crearemos un elemento div, le asignaremos un color de fondo verde oscuro, un color de texto blanco y un padding de 10 pixeles. No es importante entender como funciona el código en CSS, sino observar que se accede al elemento mediante el nombre de su clase.


<!--Creamos un elemento div y le asignamos una clase llamada color-->

<div class="color">
Contenido
</div>
/*Creamos el cuerpo de la clase color*/

.color {
background-color:darkgreen;
color:white;
padding:10px;
}

Contenido

Es importante notar como accedemos al elemento div mediante su clase "color" y podemos modificarlo visualmente desde el código en CSS.

Clases en HTML/JavaScript

En este caso, se realiza un Script que oculte el contenido de los elementos que contengan la clase "ocultar". Para ello pulsamos un boton que accione el Script. Al igual que en el caso anterior, no es importante entender como funciona el código en Java Script, sino observar que se accede al elemento mediante el nombre de su clase.


Al presionar el boton, se ejecuta la función "funciónOcultar" dentro del Script que contenga el código de Java Script
<button onclick="funcionOcultar();">
Ocultar elementos
</button>

<p class="ocultar">

function funcionOcultar() {
var x = document.getElementsByClassName("ocultar");
for(var i = 0; i < x.length; i++) {
x[i].style.display = "none";
}
}

Clases múltiples

Los elementos HTML pueden contener mas de una clase.

Para definir más de una clase, se separan los nombres de las clases con un espacio. El elemento definido de esta forma tomara las propiedades de cada una de las clases a las que pertenezca.

<div class="cubo brillante">

Sumario

  • El atributo class especifica uno o más nombres de clases para un elemento
  • Las clases se usan en CSS y JavaScript para seleccionar y acceder a elementos especificos
  • El atributo class se puede usar en cualquier elemento HTML
  • El nombre de la clase es sensible a mayusculas
  • Distintos elementos HTML pueden compartir la misma clase
  • JavaScript puede acceder a los elementos con el nombre de la clase con el metodo getElementsByClassName()
