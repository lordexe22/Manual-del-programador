Introducción

Las pseudo-clases son fragmentos de código que aplicarán estilos css en un momento o bajo acciones concretas.

Sintaxis

Para usar una pseudo clase en CSS, luego de escribir el selector se ponen : y a continuaciòn el nombre de la pseudo clase de interés. En la próxima sección se listan todas las pseudo-clases existentes.
