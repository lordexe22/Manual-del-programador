Introducción

Los backgrounds (fondos de pantalla) en CSS son propiedades que le agregan al fondo de los elemetos efectos especiales como colores e imagenes. Las propiedades que se verán nos permiten manipular el comportamiento de estas imagenes y entre otras cosas permiten crear, mezclar, posicionar y escalar las imagenes que serán asignadas a los elementos.

background-color

Permite asignarle un color de fondo a un elemento (ver sección de colores).

background-image

Esta propiedad especifica una imagen para que se use como fondo de un elemento. Por defecto, la imagen se repetira hasta que cubra el elemento completamente. La imagen en cuestión puede ser un archivo o podemos generar una imagen a partir de gradientes y otros efectos.

background-clip

Especifica si el fondo de un elemento, ya sea color o imagen, se extiende por debajo su borde.
Si la imagen o color de fondo no esta definido, esta propiedad solo tendrá un efecto visual cuando el borde tenga regiones transparentes o regiones parcialmente opacas (debido a border-style o border-image); de lo contrario el borde cubre la diferencia.

background-repeat

Esta propiedad define como se repite el fondo asignado al elemento. Un fondo de imagen puede ser repetido sobre el eje horizontal, el eje vertical, ambos ejes, o no estar repetido.

background-position

Esta propiedad define la posición de la imagen dentro del elemento. Para que su efecto sea apreciable, la imagen debe ser de menor tamaño que el disponible en el elemento y se debe controlar el como se repite la imagen dentro del elemento.

NOTA: Todos las imagenes de ejemplo de esta sección tienen la propiedad background-repeat: no-repeat;

background-blend-mode

Define como se calcula el color final de un píxel cuando hay dos capas superpuestas. Cada modo de mezcla (blend) toma el valor del color de primer plano y el de fondo (color superior y color inferior, respectivamente), realiza su cálculo y devuelve el valor de color. La capa final visible es el resultado de realizar el cálculo del modo de mezcla en cada pixel sobrepuesto entre las capas mezcladas.

background

La propiedad background es un atajo (shorthand) para definir los valores individuales del fondo en una única linea CSS. Se puede usar background para definir los valores de una o de todas las propiedades siguientes: background-attachment, color, image, position, repeat. Las mismas deben estar separadas por un espacio, a menos que se usen dos o más valores de una misma propiedad como usar lineal-gradient y url de background-image.
